# ข้อแตกแต่างระหว่าง Puts กับ Print #
## Print ##
```
int main()
{
    int x = 3;
    while (x){
        printf("---|");
        x--;
    }
}
```
` ผลคือ `
` ---|---|---| ` </br>
- ถ้าไม่ใส่ \n จะได้ผลลัพธ์ดังกล่าว

## Puts ##
```
int main()
{
    int x = 3;
    while (x){
        puts("---|");
        x--;
    }
}
```
- Puts คือมันจะปริ้นเว้นบรรทัดให้ โดยไม่ต้องใส่ \n </br>
```
ผลคือ 
---|
---|
---|
```
